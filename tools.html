Lazy tools


  • Nmap | Port scan

    nmap 127.0.0.1 -p 0-65535
    nmap -A -O -sS 127.0.0.1


  • Gobuster | Dir scan

    gobuster dir -k -u http://0.0.0.0 -w /usr/share/dirbuster/wordlists/directory-list-2.3-medium.txt -x .php,.txt,.html


  • Hydra | Brute force

    hydra -l admin -P /usr/share/wordlists/rockyou.txt host.example.com http-post-form
    "/index.php:username=^USER^&password=^PASS^&Login=Login:Login Failed"


  • Nikto | Webapp analysis

    nikto -h 127.0.0.1 -Display V | grep 200


  • Ncrack | SSH brute force

    ncrack -p 22 --user alien -P /usr/share/wordlists/rockyou.txt 127.0.0.1


  • ffuf | Subdomain scan

    ffuf -c -w /usr/share/amass/wordlists/subdomains-top1mil-110000.txt -u http://example.com/ -H "Host: FUZZ.example.com" -fs 185
